Usage

As long as the html code contain the class "typewriter" and contains the attribute "data-contents" with a provided array (similar to json format)
Example 
["Testing 1", "Testing 2", "Testing 3"]
will result in the following effect

Additional Effects

Typing Speed

By providing data-speed="x", you can increase or decrease the speed of the effect. DEFAULT_SPEED = 200

Time between each content

By providing data-period="x", you can increase or decrease the time between each content. DEFAULT_PERIOD = 2000

Randomized Contents

By providing the attribute randomized, you can randomize the list.
